02

Build log

Stage 01

Vault connected via MCP Fixed

Wired Claude Desktop to a local Obsidian vault over the filesystem MCP server. Hit a macOS permissions wall — worked around it by relocating the vault out of the Desktop folder entirely.

Stage 02

Ears — Whisper + ffmpeg Fixed

Homebrew's from-source ffmpeg build stalled over an hour on an aging Intel chip. Swapped to a prebuilt static binary. A numpy/torch version mismatch was resolved by pinning numpy to 1.x.

Stage 03

Brain — model provider Pivoted

Started with the Claude API, hit a billing checkout bug. Pivoted to Groq's free tier running Llama 3.3 70B — fast, zero cost, swappable later.

Stage 04

Voice — text-to-speech Fixed

Started with macOS's built-in say command, swapped to edge-tts for free neural voice quality — landed on en-GB-Ryan.

Stage 05

Shipped Done

All three stages wired into one script with a continuous listen-think-speak loop, plus a one-word shell alias to launch it.
